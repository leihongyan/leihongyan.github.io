	闭包的例子
### 关于this对象
```
var name='the window';
var object={
	name:'my object',
	getNameFunc:function(){
		var that=this;
		return function(){
			return that.name;
		}
	}
}

alert(object.getNameFunc()())   //my object  此处为两个括号，第一个括号为返回一个函数，第二个括号是执行该函数
```
使用that保存this变量，that引用object,所以可以返回object.getNameFunc()
